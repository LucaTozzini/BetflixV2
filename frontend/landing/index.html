Server Set Up

  1. Download backend from github repo.
  2. Make sure node is installed (tested with v18). Further instruction found here.
  3. In a terminal, navigate to "backend" folder.
  4. Install packages
    npm install
  5. Configure the .env file
    # Local
    SERVER_PORT = A_PORT_NUMBER
    DB_PATH = PATH_TO_DB # default @ /database/database.db
    MOVIES_PATH = FOLDER_WITH_MOVIES
    SHOWS_PATH = FOLDER_WITH_SHOWS

    # TMDb
    TMDB_API_KEY = YOUR_KEY
    TMDB_READ_TOKEN = YOUR_TOKEN
